Using this list

For people

Browse by topic in the sidebar. Every entry links to the issuing body's own canonical page, grouped by issuer and jurisdiction. Notes flag current versions and successors.

For AI agents

Start with llms.txt, then read manifest.json. Each entry carries routing metadata (jurisdiction, legal force, applies-to, tags, status) so you can filter to the standards that apply. It is factual metadata only, never the standard's text.
